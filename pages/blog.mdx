# Блог

Перейдите по ссылке **"Блог"** в навигационном меню: 

![search](/images/nav.png)

Произойдет переадресация на страницу блога - https://next-junkfood.vercel.app/blog

![search](/images/blog.png)

Здесь пользователь может ознакомиться с новостями компании, рецептами и другими статьями, размещенными в Блоге.
Для открытия страницы конкретной статьи достаточно кликнуть на соответствующую карточку.
